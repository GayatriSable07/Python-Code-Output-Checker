# Python Code Output Checker

## Overview

Python Code Output Checker is a GUI-based application developed using Python and Tkinter. The application allows users to solve programming problems by writing Python code and automatically checks whether the generated output matches the expected output.

## Features

* Displays programming questions.
* Accepts Python code as input.
* Executes user code.
* Compares output with expected results.
* Provides instant feedback.
* User-friendly graphical interface.

## Technologies Used

* Python
* Tkinter
* sys Module
* io Module

## How to Run

1. Download the project files.
2. Open the Python file in any Python IDE.
3. Run the program.
4. Enter your solution and click Submit Code.
